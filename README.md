# Smart Calculator

## Features
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Advanced functions (square root, exponentiation)
- User-friendly graphical interface
- History of calculations
- Scientific mode for advanced calculations

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/shaheenms257-cyber/smart-calculator.git
   ```
2. Navigate to the project directory:
   ```bash
   cd smart-calculator
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
1. Start the application:
   ```bash
   npm start
   ```
2. Use the graphical interface to input your calculations and view results.
